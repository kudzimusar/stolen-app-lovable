## ğŸ¯ Overview

**STOLEN** is a comprehensive **device registry and ownership management platform** that connects device owners, buyers, sellers, repair shops, law enforcement, insurance companies, NGOs, and financial institutions in a unified ecosystem. The platform enables users to register unlimited devices with blockchain-backed ownership records, track complete device history from purchase to disposal, transfer ownership through various methods with legal compliance, and manage device lifecycle decisions with integrated services.

## ğŸš€ Getting Started

### Prerequisites

- **Node.js** 18+ ([Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))
- **npm** 9+ or **yarn**
- **Git** for version control
- **Supabase Account** for backend services

### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd stolen-app-lovable
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**
```bash
cp .env.example .env
```

Configure the following environment variables:
```env
# Supabase
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Google Maps
VITE_GOOGLE_MAPS_API_KEY=your_google_maps_api_key

# AI Services
VITE_GEMINI_API_KEY=your_gemini_api_key
OPENAI_API_KEY=your_openai_api_key

# Payment
STRIPE_PUBLISHABLE_KEY=your_stripe_key
STRIPE_SECRET_KEY=your_stripe_secret

# Other services
CLOUDINARY_CLOUD_NAME=your_cloudinary_name
TWILIO_ACCOUNT_SID=your_twilio_sid
```

4. **Set up Supabase**
   - Create a new Supabase project
   - Run the database migrations from `database/sql/`
   - Configure Row Level Security (RLS) policies
   - Set up Edge Functions

5. **Start the development server**
```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Development Scripts

```bash
# Development
npm run dev              # Start development server
npm run dev:network      # Network access (0.0.0.0)
npm run dev:local        # Localhost only

# Building
npm run build            # Production build
npm run preview          # Preview production build

# Testing
npm run test             # Run all tests
npm run test:watch        # Watch mode
npm run test:e2e         # End-to-end tests
npm run test:coverage    # Coverage report

# Code Quality
npm run lint             # ESLint
npm run format           # Prettier
npm run type-check       # TypeScript check
```
